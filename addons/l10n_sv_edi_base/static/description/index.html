📄 El Salvador - EDI Base

Infraestructura base para facturación electrónica

🚀 Funcionalidades Principales

  • Gestión de Certificados .cert: Manejo seguro de certificados digitales del MH
  • Configuración de Ambientes: Soporte para certificación y producción
  • Integración con API MH: Preparado para comunicación con el Ministerio de Hacienda
  • Manejo de Contingencias: Soporte para modo contingencia cuando el API no está disponible
  • Numeración Automática: Generación automática de números de control DTE

📋 Documentos Soportados

  • Factura (01)
  • Comprobante de Crédito Fiscal (03)
  • Nota de Crédito (05)
  • Nota de Débito (06)
  • Y más tipos de documentos DTE

🔧 Configuración Inicial

  1. Instalar módulos de localización base (l10n_sv_cta, l10n_latam_sv)
  2. Configurar datos de la empresa (NIT, NRC, actividad económica)
  3. Cargar certificados digitales del MH
  4. Configurar ambiente (certificación/producción)
  5. Instalar módulos complementarios según necesidades

⚠️ Requisitos Importantes

  • Certificados digitales válidos del Ministerio de Hacienda
  • Registro como emisor DTE ante la DGII
  • Conexión a internet para comunicación con API del MH
  • Configuración correcta de datos fiscales de la empresa

🔗 Módulos Relacionados

  • l10n_sv_edi_json: Generación de JSON DTE
  • l10n_sv_document_type: Tipos de documentos específicos
  • l10n_sv_api_client: Cliente para API del MH
  • l10n_sv_digital_signature: Firma digital de documentos

📞 Soporte

Para soporte técnico y consultas sobre implementación, contacte a su proveedor de servicios Odoo o visite la documentación oficial del Ministerio de Hacienda de El Salvador.
